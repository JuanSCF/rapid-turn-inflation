# rapid-turn-inflation

Code files used for my thesis work CONSEQUENCES OF NON-GAUSSIANITIES ON THE ABUNDANCE OF PRIMORDIAL BLACK HOLES https://repositorio.uchile.cl/handle/2250/204025

The author is extremely grateful to Jacopo Fumagalli for extensive discussions on the subject and for sharing his version for computing f(M) in the Gaussian case. Without him, this work could not have been completed.

Brief Instructions of use:
1. f(M)-gdata-gen.py generates gaussian rapid-turn Power Spectrum and gaussian f(M)
2. dbs-g-tophat.py generates rapid-turn bi spectrum
3. xi3(M)-vec-v3.py generates the non-gaussian perturbative contribution comming from the 3-point function
4. plot-gen-v2.py integrates the non-gaussian pdf, computes and plots ng f(M) as well as the pbh abundance
